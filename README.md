# Feedback-Sorter

</div>

API для анализа тональности текстовых отзывов с поддержкой нескольких моделей машинного обучения. Проект реализует унифицированный интерфейс для обучения, оценки и развёртывания моделей, включая `классические алгоритмы` и `нейронные сети на PyTorch`.

## Обзор

**Feedback-Sorter** предоставляет REST API, который принимает текстовый ввод и возвращает предсказание тональности с оценкой уверенности и вероятностями классов. Система построена на архитектуре обёрток моделей, что обеспечивает единообразие пайплайнов обучения и инференса для алгоритмов из scikit-learn, градиентного бустинга и PyTorch. Все модели используют векторизацию TF-IDF.

### Технологический стек:

- Python 3.10+
- FastAPI + Uvicorn
- PyTorch
- scikit-learn, CatBoost, XGBoost
- joblib, pandas, numpy
- uv (менеджер пакетов и окружений)

## Использование

#### 1. Обучение моделей

* Запустите скрипт обучения

```bash
uv run python -m src.training.training_for_api
```

#### 2. Запуск API

```bash
uvicorn src.api.main:app --reload
```

API будет доступен по адресу http://127.0.0.1:8000


## Ограничения и планы на будущее

Текущая реализация демонстрирует сниженную точность при классификации отрицательной тональности. Это поведение обусловлено значительным дисбалансом классов в обучающем датасете, который содержит минимальное количество примеров с негативной окраской. В результате модели склонны смещать предсказания в сторону нейтрального или положительного класса при обработке отрицательных отзывов.

Данное ограничение будет устранено в будущих обновлениях.
